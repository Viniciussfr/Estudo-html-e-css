Tipos: Inline, Local e Externo

Inline

Aplicado indo diretamente na tag(h1,h2,p). No meio da primeira tag de algum deles se usa exemplo(<h1 style="color: ;" >)

Local

Aplicado indo no head, abaixo do title. Forma mais padrão de se usar o css, indicado quando se tem apenas 1 pagina sendo editada. Para usar é simples, <style>
body{ background-color: ;}
</style>


Externo

A forma mais efeitva quando se tem mais de 1 pagina para ser editada, é só ir em <head> e colocar um link de css, logo em seguida criar o arquivo em css e tudo que for modificado ali vai ser modificado nas paginas que se tiver o mesmo link.

Resumo

text-align= justica o texto, move para um lado e para o outro.
font-size= tamanho do texto
background-color= cor de fundo
color= cor do texto
font-family= tipo da ortografia(arial,..)
